# Privacy Policy for KeyFixer Extension

*Last Updated: December 18, 2024*

## Introduction
KeyFixer is a Chrome extension that helps users fix text typed with the wrong keyboard layout, converting between Hebrew and English. This privacy policy explains how we handle your data.

## Information Collection and Use
### What We Collect
- **Text Content**: We temporarily process the text you choose to convert
- **Website URL**: We collect the URL of the website where the conversion is being performed
- **No Personal Information**: We do not collect any personally identifiable information

### How We Use the Information
The collected information is used solely for:
- Converting text between Hebrew and English keyboard layouts
- Ensuring proper functionality of the extension

## Data Storage
- We do not store any user data
- Text processing is performed in real-time
- All data is immediately deleted after conversion
- No history or logs are maintained

## Data Transfer
- Data is transmitted securely to our server on Google Cloud Platform
- All communication is encrypted using HTTPS
- We do not share any data with third parties
- The server is located in Google's EU data center

## Security
We implement several security measures:
- Secure HTTPS encryption for all data transfers
- Cross-Origin Resource Sharing (CORS) protection
- Server security maintained by Google Cloud Platform
- Regular security updates and monitoring

## User Rights
As we don't store any personal data, there is no need for:
- Data deletion requests
- Data access requests
- Data modification requests

Users can uninstall the extension at any time to stop all data processing.

## Age Restrictions
This service is intended for users aged 13 and above.

## Changes to This Policy
We may update this privacy policy from time to time. We will notify users of any changes by updating the "Last Updated" date at the top of this policy.
